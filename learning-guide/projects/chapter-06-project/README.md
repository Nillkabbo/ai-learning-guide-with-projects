# Chapter 6 Project: IoT Fleet Management System with Visual Diagnostics

## Project Overview

Build a complete IoT fleet management system using Claude API. The system will analyze device logs, use tools to control devices, and perform visual diagnostics on device images. This demonstrates Claude's strengths in document analysis, tool use, and vision.

## Learning Objectives

- Use Claude's dedicated system parameter
- Analyze long device logs
- Implement tool use for device control
- Perform visual diagnostics
- Build a complete fleet management system

## Project Type

**Cumulative** - Builds on previous chapters, demonstrates Claude-specific features.

## Prerequisites

- Completed Chapter 6
- Anthropic API key
- Understanding of tool use
- Basic image processing knowledge

## Project Requirements

### Core Features

1. **Fleet Manager System Prompt**
   - Detailed Claude system prompt
   - IoT expert persona
   - Diagnostic procedures

2. **Device Log Analysis**
   - Read and analyze device logs
   - Long context handling
   - Pattern recognition

3. **Device Control Tools**
   - Tool use for device actions
   - Restart, configure, monitor
   - Safe tool execution

4. **Visual Diagnostics**
   - Analyze device images
   - Identify issues visually
   - Combine with log analysis

5. **Fleet Dashboard**
   - Device status overview
   - Alert system
   - Action recommendations

## Project Milestones

1. Basic Claude integration (30 min)
2. System prompt and persona (45 min)
3. Log analysis (1 hour)
4. Tool use implementation (1.5 hours)
5. Vision diagnostics (1 hour)
6. Complete integration (1 hour)

## Success Criteria

- ✅ Claude system prompts work effectively
- ✅ Long logs analyzed correctly
- ✅ Tool use functions properly
- ✅ Vision diagnostics accurate
- ✅ Complete system functional
