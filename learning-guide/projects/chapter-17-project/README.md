# Chapter 17 Project: AI-Powered IoT Dashboard

## Project Overview

Build a complete web-based IoT dashboard with AI capabilities. Includes device management, real-time monitoring, AI analysis, file uploads for diagnostics, and streaming responses for live updates.

## Learning Objectives

- Build web APIs with Flask/FastAPI
- Handle file uploads for multimodal AI
- Implement streaming responses
- Use Server-Sent Events
- Validate with Pydantic
- Build complete web application

## Project Type

**Cumulative** - Builds on all previous chapters, demonstrates web application patterns.

## Prerequisites

- Completed Chapter 17
- Basic web development knowledge
- Understanding of REST APIs
- Basic async knowledge helpful

## Project Requirements

### Core Features

1. **Web API**
   - REST endpoints
   - Device management
   - AI analysis
   - Status updates

2. **File Uploads**
   - Image diagnostics
   - PDF manuals
   - Data files
   - Validation

3. **Streaming**
   - Real-time AI responses
   - Progress updates
   - Live data
   - Better UX

4. **SSE**
   - Live device updates
   - Real-time alerts

   - Event streaming

5. **Validation**
   - Pydantic models
   - Request validation
   - Response validation
   - Error handling

6. **Dashboard UI**
   - Device list
   - Real-time status
   - AI insights
   - File uploads
