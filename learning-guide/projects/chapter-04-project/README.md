# Chapter 4 Project: Safe IoT Assistant

## Project Overview

Build a safe AI assistant that implements safeguards from Chapter 4. The assistant should classify questions, handle math safely, verify information, and demonstrate responsible AI use.

## Learning Objectives

- Implement question classification
- Handle math problems safely
- Add confidence scoring
- Build verification layers
- Create a safe assistant class

## Project Requirements

### Core Features

1. **Question Classification**
   - Detect math questions
   - Detect real-time information requests
   - Classify general questions

2. **Safe Math Handling**
   - Generate code for math problems
   - Execute code safely (with warnings)
   - Never trust AI math directly

3. **Confidence Scoring**
   - Request confidence levels from AI
   - Display confidence to users
   - Warn on low confidence

4. **Verification Layer**
   - Identify claims needing verification
   - Flag potentially hallucinated content
   - Provide verification guidance

5. **Safe Assistant Class**
   - Integrate all safeguards
   - Route questions appropriately
   - Provide safe responses

## Project Milestones

1. Basic classification (30 min)
2. Safe math handler (45 min)
3. Confidence scoring (30 min)
4. Verification layer (45 min)
5. Complete integration (1 hour)

## Success Criteria

- Classifies questions correctly
- Handles math safely
- Provides confidence scores
- Flags verification needs
- Demonstrates responsible AI use
